클래스는 기본적으로 구조체와 같다. 다만 함수를 Class안에 선언할 수 있다.

-선언

Class 클래스이름
{
...
}
으로 선언한다.

Class에서는 정보를 은닉할 수 있는데

private:
	...
를 써주면 이 클래스 밖에서 이 정보를 호출할 수 없다.
안전한 코딩을 위해서 쓰인다. 오직 이 클래스 내의 함수로만 호출이 가능하도록.

public:
	...
를 써주면 공개하겠단 것이다. 어디서든 호출이 가능하다. 기본적으로 아무 표시가 없다면 class형은 public이 써져 있는 것이다.

(struct형은 반대였던 것 같다. 기억이 가물가물가물치.)

-사용

클래스이름 만들클래스이름;
ex) Car이란 클래스의 형식으로 car1을 만들고싶다면 Car car1;

클래스안의 함수를 이용하고 싶다면

만든클래스의이름.함수();
ex) car1.Speedup();
